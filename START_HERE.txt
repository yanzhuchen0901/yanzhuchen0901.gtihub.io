💾 数据管理
================================================================================

数据存储位置：浏览器 LocalStorage（完全本地）
数据查看方法：打开浏览器 F12 → Application → LocalStorage → 你的网站
数据持久化：自动保存（无需额外操作）

关键键名：
  • daily_record_YYYY-MM-DD ... 每天的记录
  • quicknav_links_v1 ......... 快速导航链接

数据备份（强烈建议定期备份）：
  1. 打开浏览器控制台（F12）
  2. 在 Console 标签输入：exportAllRecordsAsJSON()
  3. 浏览器会自动下载 JSON 备份文件

数据恢复：
  • 在控制台输入：importRecordsFromJSON(file)
  • 选择之前备份的 JSON 文件

⚠️  重要提示
================================================================================

LocalStorage 特性：
  • 数据仅在当前浏览器中保存
  • 隐私模式不会持久化数据
  • 清空浏览器缓存会丢失数据
  • 不同浏览器/设备数据不同步

建议做法：
  • 定期备份数据（每周导出一次）
  • 使用普通浏览窗口，避免隐私模式
  • 不要清空浏览器重要数据
  • 在切换浏览器前导出数据

多设备使用：
  • 方案 1：手动导入导出（简单）
  • 方案 2：修改代码集成云同步（高级）
  • 方案 3：在每个设备上独立使用
